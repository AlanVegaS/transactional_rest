# 1. Create .env
```
# Database Configuration
POSTGRES_USER=postgres
POSTGRES_PASSWORD=postgres
POSTGRES_DB=prueba_legalario
POSTGRES_HOST=db
POSTGRES_PORT=5432

# Redis Configuration
REDIS_HOST=redis
REDIS_PORT=6379

# Gemini API Key
GEMINI_API_KEY=your-api-key
```

# 2. Build and run project
```
docker compose up --build
```

# 3. Run project
```
docker compose up
```
